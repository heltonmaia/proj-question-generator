#### Enunciado

Desenvolva um programa em Python que seja capaz de identificar o menor e o maior valor entre três números fornecidos pelo usuário. Para isso, o programa deve ser estruturado utilizando funções, promovendo a modularização e a reutilização de código, conforme os conceitos de funções e modularização apresentados no material de estudo.

Sua solução deve incluir as seguintes funções:
1.  `ler_numeros()`: Esta função deve ser responsável por solicitar ao usuário que insira três números (podendo ser inteiros ou decimais) e retornar esses três valores.
2.  `encontrar_menor_maior(num1, num2, num3)`: Esta função receberá três números como parâmetros e deverá determinar qual deles é o menor e qual é o maior. Ela deve retornar uma tupla contendo o menor e o maior valor, nessa ordem.

No programa principal, você deve chamar a função `ler_numeros()` para obter as entradas do usuário, passar esses valores para a função `encontrar_menor_maior()` para realizar a lógica e, por fim, exibir os resultados de forma clara.

#### Code Solução

```python
def ler_numeros() -> tuple[float, float, float]:
    """
    Lê três números (podendo ser inteiros ou decimais) da entrada padrão.

    Retorna:
        tuple[float, float, float]: Uma tupla contendo os três números lidos.
    """
    num1 = float(input())
    num2 = float(input())
    num3 = float(input())
    return num1, num2, num3

def encontrar_menor_maior(a: float, b: float, c: float) -> tuple[float, float]:
    """
    Encontra o menor e o maior valor entre três números dados.

    Args:
        a (float): O primeiro número.
        b (float): O segundo número.
        c (float): O terceiro número.

    Retorna:
        tuple[float, float]: Uma tupla contendo o menor e o maior valor, nessa ordem.
    """
    menor = min(a, b, c)
    maior = max(a, b, c)
    return menor, maior

# Programa principal
num1, num2, num3 = ler_numeros()
menor_valor, maior_valor = encontrar_menor_maior(num1, num2, num3)

print(f"O menor número é: {menor_valor}")
print(f"O maior número é: {maior_valor}")
```
