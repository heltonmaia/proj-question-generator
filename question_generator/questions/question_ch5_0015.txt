### Enunciado

Desenvolva uma função Python que seja capaz de analisar um conjunto de notas de um aluno. A função deve receber uma lista de números decimais (as notas) como argumento. Sua tarefa é implementar essa função para que ela calcule e retorne dois valores importantes: a soma total de todas as notas e a média aritmética dessas notas.

No programa principal, você deve:
1. Solicitar ao usuário que insira as notas do aluno em uma única linha, separadas por espaços (ex: "7.5 8.0 6.0 9.5").
2. Converter a entrada do usuário em uma lista de números decimais (`float`).
3. Chamar a função que você criou, passando a lista de notas.
4. Exibir a soma e a média calculadas, formatando ambos os resultados para duas casas decimais.

Certifique-se de que sua função retorne uma tupla contendo a soma e a média, e que o programa principal desempacote esses valores corretamente para exibição. Considere o caso de uma lista de notas vazia, retornando 0.0 para soma e média.

### Code Solução

```python
def calcular_estatisticas_notas(notas: list[float]) -> tuple[float, float]:
    """
    Calcula a soma e a média de uma lista de notas.

    Args:
        notas (list[float]): Uma lista de notas.

    Returns:
        tuple[float, float]: Uma tupla contendo a soma das notas e a média das notas.
                             Retorna (0.0, 0.0) se a lista estiver vazia.
    """
    if not notas:
        return 0.0, 0.0
    
    soma_total = sum(notas)
    media_aritmetica = soma_total / len(notas)
    return soma_total, media_aritmetica

# Programa principal
entrada_notas = input()
# Divide a string de entrada por espaços e converte cada parte para float
notas_str = entrada_notas.split()
notas_float = [float(nota) for nota in notas_str]

# Chama a função e desempacota os múltiplos valores de retorno
soma_resultado, media_resultado = calcular_estatisticas_notas(notas_float)

# Exibe os resultados formatados para duas casas decimais
print(f"Soma: {soma_resultado:.2f}")
print(f"Média: {media_resultado:.2f}")
```
