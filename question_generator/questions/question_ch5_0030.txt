#### Enunciado

Desenvolva um programa em Python que determine o menor e o maior número entre três valores fornecidos pelo usuário. O programa deve utilizar funções para organizar e modularizar o código, conforme as seguintes instruções:

1.  **Função para Leitura de Números:** Crie uma função chamada `ler_numeros()` que não recebe parâmetros. Esta função deve solicitar ao usuário que insira três números (um por linha) e retornar esses três valores como uma tupla de floats.
2.  **Função para Encontrar Menor e Maior:** Implemente uma função chamada `encontrar_menor_maior(a, b, c)` que receba três números como parâmetros. Esta função deve identificar o menor e o maior valor entre eles e retornar uma tupla contendo o menor e o maior número, nesta ordem.
3.  **Programa Principal:** No corpo principal do programa, utilize as funções criadas para:
    *   Obter os três números do usuário chamando `ler_numeros()`.
    *   Calcular o menor e o maior valor entre os números lidos, chamando `encontrar_menor_maior()`.
    *   Exibir os resultados na tela, informando qual é o menor e qual é o maior número.

**Observação:** O programa deve ser capaz de tratar números inteiros e decimais (ponto flutuante).

#### Code Solução

```python
def ler_numeros() -> tuple[float, float, float]:
    """
    Solicita ao usuário que insira três números e os retorna como uma tupla.
    Cada número é lido de uma nova linha.
    """
    num1 = float(input())
    num2 = float(input())
    num3 = float(input())
    return num1, num2, num3

def encontrar_menor_maior(a: float, b: float, c: float) -> tuple[float, float]:
    """
    Recebe três números como parâmetros e retorna uma tupla contendo
    o menor e o maior valor entre eles, nesta ordem.
    """
    menor = min(a, b, c)
    maior = max(a, b, c)
    return menor, maior

# Programa principal
num1, num2, num3 = ler_numeros()
menor, maior = encontrar_menor_maior(num1, num2, num3)

print(f"O menor número é: {menor}")
print(f"O maior número é: {maior}")
```
