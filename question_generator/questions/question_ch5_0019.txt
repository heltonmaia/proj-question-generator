#### Enunciado

Desenvolva um programa em Python que analise uma lista de números inteiros fornecida pelo usuário, calculando e exibindo as seguintes estatísticas: a soma total dos números, a média aritmética e a contagem de números pares e ímpares.

Para isso, seu programa deve seguir as seguintes instruções:

1.  **Entrada de Dados:** O programa deve solicitar ao usuário uma única linha de texto contendo números inteiros separados por espaços. Estes números devem ser lidos e convertidos em uma lista de inteiros.
2.  **Função `analisar_estatisticas`:** Crie uma função chamada `analisar_estatisticas` que aceite uma lista de números inteiros como parâmetro. Esta função deve calcular a soma de todos os elementos da lista e a média desses elementos. Ela deve retornar esses dois valores em uma tupla (soma, média). Se a lista estiver vazia, a soma deve ser 0 e a média 0.0.
3.  **Função `contar_par_impar`:** Crie uma função chamada `contar_par_impar` que também aceite uma lista de números inteiros como parâmetro. Esta função deve percorrer a lista e contar quantos números são pares e quantos são ímpares. Ela deve retornar essas duas contagens em uma tupla (pares, ímpares).
4.  **Programa Principal:** No programa principal, após obter a lista de números do usuário, chame as funções `analisar_estatisticas` e `contar_par_impar`. Em seguida, imprima os resultados de forma clara e formatada, como mostrado nos exemplos de saída. A média deve ser formatada com duas casas decimais.

#### Code Solução

```python
def analisar_estatisticas(numeros: list[int]) -> tuple[int, float]:
    """
    Calcula a soma e a média de uma lista de números inteiros.

    Args:
        numeros: Uma lista de números inteiros.

    Returns:
        Uma tupla contendo a soma (int) e a média (float) dos números.
        Retorna (0, 0.0) se a lista for vazia.
    """
    if not numeros:
        return 0, 0.0
    
    soma = sum(numeros)
    media = soma / len(numeros)
    return soma, media

def contar_par_impar(numeros: list[int]) -> tuple[int, int]:
    """
    Conta a quantidade de números pares e ímpares em uma lista.

    Args:
        numeros: Uma lista de números inteiros.

    Returns:
        Uma tupla contendo a contagem de números pares e a contagem de números ímpares.
    """
    pares = 0
    impares = 0
    for num in numeros:
        if num % 2 == 0:
            pares += 1
        else:
            impares += 1
    return pares, impares

# Leitura da entrada
entrada_usuario_str = input() 
lista_de_numeros = [int(x) for x in entrada_usuario_str.split()]

# Chamada das funções
soma_total, media_calculada = analisar_estatisticas(lista_de_numeros)
pares_contados, impares_contados = contar_par_impar(lista_de_numeros)

# Impressão dos resultados
print(f"Soma dos números: {soma_total}")
print(f"Média dos números: {media_calculada:.2f}")
print(f"Quantidade de números pares: {pares_contados}")
print(f"Quantidade de números ímpares: {impares_contados}")
```
