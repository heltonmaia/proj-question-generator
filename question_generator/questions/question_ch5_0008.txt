#### Enunciado

Desenvolva uma função Python chamada `analisar_desempenho_notas` que receba uma lista de notas (números de ponto flutuante) de um estudante. A função deve calcular a média aritmética dessas notas e, em seguida, determinar quantos alunos tiveram uma nota estritamente superior à média calculada.

A função deve retornar dois valores:
1.  A média das notas (um número de ponto flutuante).
2.  A quantidade de notas que são estritamente maiores que a média (um número inteiro).

Certifique-se de incluir `docstrings` e `type hints` para a função, conforme as boas práticas de Python. Caso a lista de notas esteja vazia, a função deve retornar `(0.0, 0)`.

#### Solução

```python
def analisar_desempenho_notas(notas: list[float]) -> tuple[float, int]:
    """
    Analisa uma lista de notas, calculando a média e a quantidade de notas acima da média.

    Args:
        notas (list[float]): Uma lista de notas de ponto flutuante.

    Returns:
        tuple[float, int]: Uma tupla contendo a média das notas e a quantidade de notas
                           estritamente acima da média. Retorna (0.0, 0) se a lista estiver vazia.
    """
    if not notas:
        return 0.0, 0

    soma_notas = sum(notas)
    media = soma_notas / len(notas)

    notas_acima_media = 0
    for nota in notas:
        if nota > media:
            notas_acima_media += 1
            
    return media, notas_acima_media
```
