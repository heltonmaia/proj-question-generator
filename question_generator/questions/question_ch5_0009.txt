#### Enunciado

Crie um programa Python que funcione como uma calculadora de área para diferentes formas geométricas: triângulo, círculo e retângulo. O programa deve interagir com o usuário através de um menu, permitindo a escolha da forma e a inserção das dimensões.

Você deve implementar as seguintes funções:

*   `area_triangulo(base: float, altura: float) -> float`: Calcula a área de um triângulo. A fórmula é `(base * altura) / 2`.
*   `area_circulo(raio: float) -> float`: Calcula a área de um círculo. A fórmula é `π * (raio ** 2)`. Utilize `math.pi` para o valor de π.
*   `area_retangulo(comprimento: float, largura: float) -> float`: Calcula a área de um retângulo. A fórmula é `comprimento * largura`.

A função principal do programa, `calcular_area_menu()`, deve apresentar um menu ao usuário com as seguintes opções:
1.  Calcular a área de um triângulo
2.  Calcular a área de um círculo
3.  Calcular a área de um retângulo
4.  Sair do programa

O programa deve continuar exibindo o menu e processando as escolhas até que o usuário selecione a opção '4' (Sair). Para cada cálculo, valide as entradas para garantir que as dimensões (base, altura, raio, comprimento, largura) sejam números positivos. Se uma dimensão inválida for inserida (negativa ou zero), exiba uma mensagem de erro específica e não realize o cálculo. Trate também erros de entrada onde o usuário digita algo que não é um número.

Todos os resultados de área devem ser arredondados para duas casas decimais.

#### Code Solução

```python
import math

def area_triangulo(base: float, altura: float) -> float:
    """
    Calcula a área de um triângulo.

    Args:
        base (float): A base do triângulo.
        altura (float): A altura do triângulo.

    Returns:
        float: A área calculada do triângulo.
    """
    return (base * altura) / 2

def area_circulo(raio: float) -> float:
    """
    Calcula a área de um círculo.

    Args:
        raio (float): O raio do círculo.

    Returns:
        float: A área calculada do círculo.
    """
    return math.pi * (raio ** 2)

def area_retangulo(comprimento: float, largura: float) -> float:
    """
    Calcula a área de um retângulo.

    Args:
        comprimento (float): O comprimento do retângulo.
        largura (float): A largura do retângulo.

    Returns:
        float: A área calculada do retângulo.
    """
    return comprimento * largura

def calcular_area_menu() -> None:
    """
    Função principal que apresenta um menu para o usuário para calcular
    áreas de diferentes formas geométricas.
    """
    while True:
        print("\n--- Calculadora de Área de Formas Geométricas ---")
        print("1. Calcular a área de um triângulo")
        print("2. Calcular a área de um círculo")
        print("3. Calcular a área de um retângulo")
        print("4. Sair do programa")

        escolha = input("Escolha uma opção: ")

        if escolha == '1':
            try:
                base = float(input("Digite a base do triângulo: "))
                altura = float(input("Digite a altura do triângulo: "))
                if base <= 0 or altura <= 0:
                    print("Erro: A base e a altura devem ser valores positivos.")
                else:
                    area = area_triangulo(base, altura)
                    print(f"A área do triângulo é: {area:.2f}")
            except ValueError:
                print("Entrada inválida. Por favor, digite um número.")
        elif escolha == '2':
            try:
                raio = float(input("Digite o raio do círculo: "))
                if raio <= 0:
                    print("Erro: O raio deve ser um valor positivo.")
                else:
                    area = area_circulo(raio)
                    print(f"A área do círculo é: {area:.2f}")
            except ValueError:
                print("Entrada inválida. Por favor, digite um número.")
        elif escolha == '3':
            try:
                comprimento = float(input("Digite o comprimento do retângulo: "))
                largura = float(input("Digite a largura do retângulo: "))
                if comprimento <= 0 or largura <= 0:
                    print("Erro: O comprimento e a largura devem ser valores positivos.")
                else:
                    area = area_retangulo(comprimento, largura)
                    print(f"A área do retângulo é: {area:.2f}")
            except ValueError:
                print("Entrada inválida. Por favor, digite um número.")
        elif escolha == '4':
            print("Programa encerrado.")
            break
        else:
            print("Opção inválida. Por favor, escolha uma opção de 1 a 4.")

# Inicia o programa
if __name__ == "__main__":
    calcular_area_menu()
```
