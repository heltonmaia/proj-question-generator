### Enunciado

O objetivo é criar um programa que permita ao usuário transformar uma lista predefinida de números inteiros aplicando diferentes operações matemáticas. Você deverá implementar uma função principal que gerencie a seleção da operação e utilize funções auxiliares para realizar as transformações.

1.  **Função `transformar_lista(lista_numeros: list[int], operacao)`:**
    *   Esta função deve receber uma lista de números inteiros (`lista_numeros`) e uma função (`operacao`) como argumentos.
    *   Ela deve aplicar a `operacao` a cada elemento da `lista_numeros`.
    *   A função deve retornar uma **nova lista** contendo os resultados das transformações.

2.  **Funções de Operação (pré-definidas):**
    *   Implemente três funções separadas para as operações básicas:
        *   `dobrar(numero: int) -> int`: Retorna o dobro do número.
        *   `quadrado(numero: int) -> int`: Retorna o quadrado do número.
        *   `triplicar_lambda = lambda x: x * 3`: Uma função lambda (anônima) que retorna o triplo do número.

3.  **Programa Principal:**
    *   Defina uma lista de números inteiros inicial (ex: `[1, 2, 3, 4, 5]`).
    *   Apresente um menu ao usuário para escolher a operação desejada:
        *   `1 - Dobrar números`
        *   `2 - Calcular o quadrado dos números`
        *   `3 - Triplicar números`
        *   `4 - Sair`
    *   Com base na escolha do usuário, chame a função `transformar_lista` passando a lista inicial e a função de operação correspondente.
    *   Imprima a lista original e a lista transformada.
    *   O programa deve continuar exibindo o menu e processando escolhas até que o usuário selecione a opção "Sair".
    *   Valide a entrada do usuário para a escolha do menu, informando quando a opção for inválida.

**Conceitos abordados:** Definição de funções, funções com retorno, parâmetros posicionais, funções que recebem outras funções como argumentos, funções anônimas (lambda), listas, laços de repetição (`while`), condicionais (`if/elif/else`), entrada/saída de dados.

#### Code Solução

```python
# Funções de Operação
def dobrar(numero: int) -> int:
    """Retorna o dobro de um número."""
    return numero * 2

def quadrado(numero: int) -> int:
    """Retorna o quadrado de um número."""
    return numero ** 2

# Função lambda para triplicar
triplicar_lambda = lambda x: x * 3

def transformar_lista(lista_numeros: list[int], operacao) -> list[int]:
    """
    Aplica uma operação a cada elemento de uma lista de números.

    Args:
        lista_numeros (list[int]): A lista de números a ser transformada.
        operacao (function): A função a ser aplicada a cada número.

    Returns:
        list[int]: Uma nova lista com os resultados da operação.
    """
    return [operacao(num) for num in lista_numeros]

# Definição da lista original
numeros_originais = [1, 2, 3, 4, 5]

# Loop principal para o menu
while True:
    print("\n--- Transformador de Listas Numéricas ---")
    print(f"Lista original: {numeros_originais}")
    print("Escolha uma operação:")
    print("1 - Dobrar números")
    print("2 - Calcular o quadrado dos números")
    print("3 - Triplicar números")
    print("4 - Sair")
    
    escolha = input("Digite sua opção: ")
    
    if escolha == '1':
        lista_transformada = transformar_lista(numeros_originais, dobrar)
        print(f"Lista dobrada: {lista_transformada}")
    elif escolha == '2':
        lista_transformada = transformar_lista(numeros_originais, quadrado)
        print(f"Lista com quadrados: {lista_transformada}")
    elif escolha == '3':
        lista_transformada = transformar_lista(numeros_originais, triplicar_lambda)
        print(f"Lista triplicada: {lista_transformada}")
    elif escolha == '4':
        print("Programa encerrado.")
        break
    else:
        print("Opção inválida. Por favor, tente novamente.")
```
