#### Enunciado

Desenvolva uma função Python chamada `analisar_numeros` que receba uma lista de números inteiros como argumento. A função deve calcular e retornar os seguintes resultados em uma única tupla, na ordem especificada:

1.  A soma de todos os números na lista.
2.  A quantidade de números pares.
3.  A quantidade de números ímpares.
4.  O menor número na lista.
5.  O maior número na lista.

Se a lista de entrada estiver vazia, a soma e as contagens devem ser zero, e o menor e o maior número devem ser retornados como `None`.

No programa principal, você deve:
*   Ler uma linha de números inteiros separados por espaço.
*   Converter esses números em uma lista de inteiros.
*   Chamar a função `analisar_numeros` com essa lista.
*   Imprimir os resultados obtidos, formatando-os conforme os exemplos de saída.

#### Code Solução

```python
def analisar_numeros(lista_de_numeros: list[int]) -> tuple[int, int, int, int | None, int | None]:
    """
    Analisa uma lista de números inteiros, retornando a soma, contagem de pares e ímpares,
    e o menor e maior número.

    Args:
        lista_de_numeros: Uma lista de números inteiros.

    Returns:
        Uma tupla contendo (soma, contagem_pares, contagem_impares, menor_numero, maior_numero).
        Se a lista estiver vazia, menor_numero e maior_numero serão None.
    """
    if not lista_de_numeros:
        return 0, 0, 0, None, None

    soma_total = sum(lista_de_numeros)
    menor_val = min(lista_de_numeros)
    maior_val = max(lista_de_numeros)
    
    pares = 0
    impares = 0
    for num in lista_de_numeros:
        if num % 2 == 0:
            pares += 1
        else:
            impares += 1

    return soma_total, pares, impares, menor_val, maior_val

# Leitura da entrada (os números são lidos como uma string e convertidos para lista de inteiros)
entrada_str = input()
if entrada_str:
    lista_numeros_str = entrada_str.split()
    lista_numeros_int = [int(n) for n in lista_numeros_str]
else:
    lista_numeros_int = []

# Chamada da função e impressão dos resultados
soma, pares, impares, menor, maior = analisar_numeros(lista_numeros_int)

print(f"Soma total: {soma}")
print(f"Quantidade de pares: {pares}")
print(f"Quantidade de ímpares: {impares}")
if menor is not None:
    print(f"Menor número: {menor}")
    print(f"Maior número: {maior}")
else:
    print("Lista vazia: não há menor ou maior número.")
```
