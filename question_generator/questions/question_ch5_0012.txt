#### Enunciado

Desenvolva um programa Python que permita ao usuário inserir uma sequência de números e, em seguida, calcule e exiba algumas estatísticas básicas: a média, o menor valor e o maior valor dessa sequência. O programa deve utilizar funções para organizar o código e demonstrar o uso de funções com múltiplos valores de retorno.

Siga as seguintes instruções:

1.  **Função para Leitura de Dados (`ler_lista_numeros`)**:
    *   Crie uma função chamada `ler_lista_numeros` que não recebe parâmetros.
    *   Essa função deve solicitar ao usuário que digite uma série de números, separados por espaços, em uma única linha (ex: `10 20.5 5 12.8`).
    *   A função deve converter esses números em uma lista de valores do tipo `float` e retorná-la.
    *   **Observação:** Se o usuário não digitar nada (apenas Enter) ou apenas espaços, a função deve retornar uma lista vazia.

2.  **Função para Análise de Estatísticas (`analisar_estatisticas`)**:
    *   Crie uma função chamada `analisar_estatisticas` que recebe como parâmetro uma lista de números (`list[float]`).
    *   Essa função deve calcular a **média**, o **menor valor** e o **maior valor** presentes na lista.
    *   A função deve retornar esses três valores em uma única tupla, na ordem: `(media, menor_valor, maior_valor)`.
    *   Inclua uma `docstring` e `type hints` para a função.
    *   **Importante:** Assuma que a lista passada para esta função não estará vazia para os cálculos de média, mínimo e máximo.

3.  **Programa Principal**:
    *   No programa principal, chame a função `ler_lista_numeros` para obter a lista de números do usuário.
    *   Verifique se a lista retornada está vazia. Se estiver, imprima uma mensagem informando que nenhum número foi inserido e que as estatísticas não podem ser calculadas.
    *   Se a lista não estiver vazia, chame a função `analisar_estatisticas` passando a lista obtida.
    *   Desempacote os resultados retornados pela `analisar_estatisticas` em variáveis apropriadas.
    *   Imprima os resultados formatando a média, o menor valor e o maior valor com **duas casas decimais**.

#### Code Solução

```python
def ler_lista_numeros() -> list[float]:
    """
    Solicita ao usuário uma lista de números e os converte para float.
    Os números devem ser inseridos na mesma linha, separados por espaços.
    Se nenhuma entrada for fornecida ou apenas espaços, retorna uma lista vazia.
    """
    entrada_str = input("Digite os números separados por espaços (ex: 10 20.5 5 12.8): ")
    
    # Filtra strings vazias resultantes de múltiplos espaços ou entrada vazia
    numeros_str = [num for num in entrada_str.split(' ') if num.strip()]
    
    if not numeros_str:
        return []
    
    return [float(num) for num in numeros_str]

def analisar_estatisticas(numeros: list[float]) -> tuple[float, float, float]:
    """
    Calcula a média, o valor mínimo e o valor máximo de uma lista de números.

    Args:
        numeros (list[float]): A lista de números para análise.
                               Assumido que a lista não estará vazia.

    Returns:
        tuple[float, float, float]: Uma tupla contendo (media, minimo, maximo).
    """
    soma_total = sum(numeros)
    quantidade = len(numeros)
    
    media = soma_total / quantidade
    menor_valor = min(numeros)
    maior_valor = max(numeros)
    
    return media, menor_valor, maior_valor

# --- Programa Principal ---
if __name__ == "__main__":
    lista_de_numeros = ler_lista_numeros()

    if not lista_de_numeros:
        print("Nenhum número foi inserido. Não é possível calcular estatísticas.")
    else:
        media, menor, maior = analisar_estatisticas(lista_de_numeros)
        print(f"Média: {media:.2f}")
        print(f"Menor valor: {menor:.2f}")
        print(f"Maior valor: {maior:.2f}")

```
