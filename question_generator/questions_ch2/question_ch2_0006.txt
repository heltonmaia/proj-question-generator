#### Enunciado

O programa deve solicitar ao usuário que digite cinco números inteiros, um por vez. Para cada número, o programa deve verificar se ele é par ou ímpar e imprimir a mensagem correspondente. A verificação para cada número deve ser feita independentemente, sem o uso de laços de repetição (como `for` ou `while`), utilizando apenas estruturas condicionais (`if-else`).

#### Code Solução

```python
# Primeiro número
num1 = int(input())
if num1 % 2 == 0:
    print(f"O número {num1} é par.")
else:
    print(f"O número {num1} é ímpar.")

# Segundo número
num2 = int(input())
if num2 % 2 == 0:
    print(f"O número {num2} é par.")
else:
    print(f"O número {num2} é ímpar.")

# Terceiro número
num3 = int(input())
if num3 % 2 == 0:
    print(f"O número {num3} é par.")
else:
    print(f"O número {num3} é ímpar.")

# Quarto número
num4 = int(input())
if num4 % 2 == 0:
    print(f"O número {num4} é par.")
else:
    print(f"O número {num4} é ímpar.")

# Quinto número
num5 = int(input())
if num5 % 2 == 0:
    print(f"O número {num5} é par.")
else:
    print(f"O número {num5} é ímpar.")
```
