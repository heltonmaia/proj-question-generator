#### Enunciado

Crie um programa para gerenciar uma lista de compras básica.

1.  Inicialize uma lista vazia chamada `lista_compras`.
2.  Solicite ao usuário que insira **três itens** de compras, um por um, e adicione-os à `lista_compras`.
3.  Após os itens serem adicionados, imprima a lista atual.
4.  Pergunte ao usuário se ele deseja **remover algum item** (a resposta deve ser 'sim' ou 'nao').
5.  Se a resposta for 'sim' (ignorando maiúsculas/minúsculas):
    *   Solicite o nome do item a ser removido.
    *   Verifique se o item a ser removido existe na `lista_compras`.
        *   Se existir, remova-o e imprima uma mensagem de confirmação.
        *   Se não existir, imprima uma mensagem informando que o item não foi encontrado.
6.  Se a resposta for 'nao', ou se for uma opção inválida, imprima uma mensagem indicando que nenhum item será removido.
7.  Finalmente, imprima o estado final da `lista_compras`.

#### Code Solução

```python
# 1. Inicialize uma lista vazia
lista_compras = []

# 2. Solicite e adicione três itens
print("Por favor, adicione três itens à sua lista de compras:")
item1 = input("Item 1: ")
lista_compras.append(item1)

item2 = input("Item 2: ")
lista_compras.append(item2)

item3 = input("Item 3: ")
lista_compras.append(item3)

# 3. Imprima a lista atual
print(f"\nSua lista de compras atual: {lista_compras}")

# 4. Pergunte sobre remoção
resposta_remocao = input("\nDeseja remover algum item da lista? (sim/nao): ").lower()

# 5 & 6. Processe a remoção
if resposta_remocao == 'sim':
    item_remover = input("Qual item você deseja remover?: ")
    if item_remover in lista_compras:
        lista_compras.remove(item_remover)
        print(f"'{item_remover}' foi removido da lista.")
    else:
        print(f"'{item_remover}' não encontrado na lista.")
elif resposta_remocao == 'nao':
    print("Nenhum item será removido.")
else:
    print("Opção inválida. Nenhum item será removido.")

# 7. Imprima o estado final da lista
print(f"\nLista de compras final: {lista_compras}")
```
